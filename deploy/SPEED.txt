Скорость 
1) Обновить nginx
   sudo cp ~/schoolgame/deploy/nginx-site.conf /etc/nginx/sites-available/pokemon-game.byred.fun
   sudo nginx -t && sudo systemctl reload nginx

2) Если nginx -t ругается на ssl_dhparam — выполни один раз:
   sudo certbot renew --dry-run
   (или закомментируй строку ssl_dhparam в vhost)

3) Пересобери Web в Godot и залей builds/web (пресет уже с for_mobile=true при необходимости).

4) Максимум нагрузки: включи Cloudflare (оранжевое облако) + Cache Rules для *.wasm *.js *.pck — см. обсуждение в чате.

Предупреждение nginx "protocol options redefined for 0.0.0.0:443" при нескольких vhost на одном IP — часто только warn; если всё работает, можно не трогать.

Реальность по скорости (важно)
========================
Сейчас в builds/web типично ~34 MB .wasm + ~72 MB .pck ≈ 100+ MB на первый заход.
Nginx gzip/preload не превратят это в «мгновенно» — нужно:
- уменьшать PCK в Godot (текстуры, лишние ресурсы, export filter),
- или Cloudflare (кэш у edge),
- или отдельный CDN/объектное хранилище под статику.
